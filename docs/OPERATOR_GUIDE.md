## 3. トラブルシューティング

| 症状 | 原因 | 対処 |
|------|------|------|
| **Graph が空・Loading** | Neo4j 未起動 | Neo4j Desktop で "Recommendation" を **Start** |
| **API エラー表示** | port 8000 使用中 | API を再起動: `py -m uvicorn api.api_server:app --port 8000` |
| **Demo Mode バナー** | Neo4j 停止中 | fallback データで完走可能。本番は Neo4j 起動推奨 |
| **Recommend 0件** | v3 オントロジー未構築 | `py graph_builder.py` を実行（5分） |
| **画面真っ白** | Next.js 未起動 | `cd demo-web && npm run dev` |

# API 確認
curl http://localhost:8000/health
# → {"status":"ok","neo4j":"connected"}

# Next.js 確認
curl http://localhost:3000
# → HTML が返る

## 5. よくある質問

**Q: Neo4j が必須ですか？**  
A: 推奨ですが、停止時は fallback データで完走できます。Graph は固定データになります。

**Q: LLM は使っていますか？**  
A: 現在はテンプレート生成。Phase 4 で Claude API 統合予定です。

**Q: データは保存されますか？**  
A: 現在はメモリのみ。Supabase 統合は Phase 4 以降です。

**Q: スマホで見れますか？**  
A: レスポンシブ対応済みですが、デモは 1920px 投影推奨です。

**Q: 7分より短くできますか？**  
A: Questions を3問に減らす、Graph アニメーションをスキップで 5分台も可能です。
